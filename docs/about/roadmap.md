# 发展规划

OneLook 的未来发展路线图。

##  当前版本 (V0.2.x)

- [x] 基础思维导图编辑
- [x] 多种布局模式（思维导图/树形/鱼骨/组织）
- [x] Markdown 与 LaTeX 支持
- [x] 多格式导出（PNG/SVG/Markdown/.olook）
- [x] 全键盘快捷键
- [x] 暗黑模式
- [x] 演说模式 (Pitch Mode)
- [x] 禅模式 (Zen Mode)
- [x] 主流格式导入导出（XMind / FreeMind / OPML）

##  近期计划 (V0.3.x)

- [ ] **AI Assistant（第一阶段）**：自然语言生成导图草稿
- [ ] **节点批处理工具**：批量重命名、批量样式与批量整理
- [ ] **大图性能优化**：超大节点数场景下的渲染与交互提速

##  中期目标 (V1.0)

- [ ] **AI 智能辅助**：接入 LLM，自然语言生成导图
- [ ] **节点扩写与总结**：AI 辅助内容创作
- [ ] **PWA 支持**：离线使用与桌面安装

##  远期愿景

- [ ] 协作编辑
- [ ] 移动端应用
- [ ] 插件生态
